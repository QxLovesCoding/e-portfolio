2025

A Raspberry Pi 4-based smart pill dispenser for assisted care patients. Features a secure 4-digit joystick passcode on SenseHat LED matrix, servo-controlled dispensing, ultrasonic pill counting, and real-time cloud monitoring via Adafruit IO.
