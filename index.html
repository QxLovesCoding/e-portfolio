PRJ-03 // EGR206 IOT PROJECT — NYP

Smart Pill Dispenser

2025
1 / 14
The Problem
50% of patients don't take medication as prescribed. Over 125,000 deaths per year are linked to medication non-adherence and 1 in 3 assisted care patients experience a dosing error. Current solutions (pill boxes, blister packs, nurse rounds) have no automated alerts, tracking or remote visibility.
Our Solution
A fully integrated, IoT-connected medication dispenser built on Raspberry Pi 4 with GrovePi+, SenseHat, servo motors and Adafruit IO. The device secures, dispenses, monitors and reports medication usage in real time — with cloud sync and a doctor-facing GUI dashboard.
Tech Stack
Raspberry Pi 4 Python Tkinter GrovePi+ SenseHat PCA9685 Servo Driver Adafruit IO I²C LCD Display
System Features
  • 4-digit joystick passcode on SenseHat LED matrix (wrong entry → red flash + scrolling alert)
  • Servo-controlled security panel — auto-relocks after 5-second buzzer warning
  • Button-triggered dispenser servo — drops one pill per press
  • Ultrasonic pill counter — maps distance to pill count (0–4), shown on LCD
  • Light sensor tamper detection — buzzer and LED alarm if panel left open
  • Debug mode — hidden passcode [0,0,0,0] for engineering access, all events logged to cloud
